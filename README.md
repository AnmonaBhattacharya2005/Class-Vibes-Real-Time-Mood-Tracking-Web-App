Class Vibes – Real-Time Mood Tracking Web App is an interactive web application that allows students to share and track their daily classroom moods in real time. Users can select emotions like happy, stressed, excited, tired, or motivated, and the system instantly updates mood statistics using colorful charts and visual analytics. The platform helps teachers and students understand the overall classroom environment and encourages better communication and engagement. Built using HTML, CSS, JavaScript, and Chart.js, the project focuses on user interaction, responsive design, and real-time data visualization.
